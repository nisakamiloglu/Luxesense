# LuxeSense-AI

LuxeSense-AI is an AI-powered luxury shopping application designed to connect customers and sales associates through intelligent recommendations and a personal AI stylist chat. 

The app aims to improve the luxury retail experience by providing personalized suggestions, real-time communication, and smart dashboards.

## Features

- AI-powered product recommendations
- Personal AI stylist chat
- Customer dashboard
- Sales associate dashboard
- Real-time interaction between customers and advisors
- Modern mobile UI built with Expo / React Native
  
## How to run the project

### 1. Clone the repository

git clone https://github.com/nisakamiloglu/LuxeSense-AI.git

### 2. Go into the project folder

cd LuxeSense-AI

### 3. Install dependencies

npm install

### 4. Start the Expo server

npx expo start

### 5. Open the app on your phone

- Install Expo Go from App Store or Google Play
- Scan the QR code shown in the terminal
- The app will open on your phone


## Requirements

- Node.js installed
- npm installed
- Expo Go installed on phone
